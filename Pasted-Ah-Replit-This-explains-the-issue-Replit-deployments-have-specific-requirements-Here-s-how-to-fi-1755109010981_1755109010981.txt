Ah, Replit! This explains the issue. Replit deployments have specific requirements. Here's how to fix it:

## Replit-Specific Solutions

### 1. Update `.replit` Configuration File
Create or update your `.replit` file in the root directory:

```toml
modules = ["nodejs-20"]

[deployment]
deploymentTarget = "static"
publicDir = "dist"
build = ["npm", "run", "build"]

[[ports]]
localPort = 5000
externalPort = 80
```

### 2. Fix Package.json for Replit
Your `package.json` looks correct, but make sure it's in the **root directory**:

```json
{
  "name": "your-replit-app",
  "version": "1.0.0",
  "scripts": {
    "dev": "tsx server/index-no-vite.ts",
    "build": "vite build",
    "start": "node server/index.js"
  },
  "dependencies": {
    "express": "^4.18.2",
    "tsx": "^4.0.0",
    "vite": "^5.4.19",
    "@vitejs/plugin-react": "^5.0.0"
  }
}
```

### 3. Create `replit.nix` File
```nix
{ pkgs }: {
  deps = [
    pkgs.nodejs_20
    pkgs.npm
  ];
}
```

### 4. Fix Directory Structure
Make sure your structure looks like this:
```
your-replit/
├── .replit
├── replit.nix
├── package.json
├── vite.config.js
├── index.html
├── src/
│   └── (your React files)
└── server/
    └── (your server files)
```

### 5. Update Vite Config
Create `vite.config.js` in root:

```javascript
import { defineConfig } from 'vite'
import react from '@vitejs/plugin-react'

export default defineConfig({
  plugins: [react()],
  build: {
    outDir: 'dist'
  },
  server: {
    host: '0.0.0.0',
    port: 5000
  }
})
```

### 6. Deployment Steps in Replit

1. **Go to the "Deploy" tab** in your Replit
2. **Choose "Static" deployment** (not Autoscale)
3. **Set build command**: `npm run build`
4. **Set public directory**: `dist`
5. **Click Deploy**

### 7. Alternative: Server-Side Deployment

If you need server-side features, use this `.replit` instead:

```toml
modules = ["nodejs-20"]

[deployment]
deploymentTarget = "autoscale"
build = ["npm", "run", "build"]
run = ["npm", "start"]

[[ports]]
localPort = 5000
externalPort = 80
```

### 8. Test Locally First
In Replit console, run:
```bash
npm install
npm run build
ls dist/  # Should show your built files
```

### Common Replit Issues:

- **Cache problems**: Delete `node_modules` and run `npm install` again
- **Wrong deployment type**: Use "Static" for SPA, "Autoscale" for full-stack
- **Port conflicts**: Make sure your server uses `process.env.PORT || 5000`
